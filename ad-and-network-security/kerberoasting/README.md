# Kerberoasting

### What It Is

Kerberoasting is an attack where any authenticated domain user requests Kerberos Service Tickets for accounts that have Service Principal Names (SPNs), then attempts to crack those tickets offline to recover the service account's password.

No elevated privileges required. No special tools during the request phase. No lockout mechanism. No time pressure on the cracking.
